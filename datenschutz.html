Diese Website wird über GitHub Pages bereitgestellt. Beim Aufruf können technisch notwendige Zugriffsdaten (z. B. IP-Adresse, Zeitpunkt, angeforderte Seite) verarbeitet werden.

Auf der Startseite wird ein Design über einen iFrame von Canva eingebettet. Dadurch wird beim Laden der Seite eine Verbindung zu Servern von Canva aufgebaut. Dabei können Daten (z. B. IP-Adresse, Geräte-/Browser-Infos) an Canva übermittelt werden.
